技术描述

本程序采用SSM(SpringMvc,SpringFramework,Mybatis)框架, Log4j日志框架,Mysql (8.0.19) 数据库 作数据存储. Maven进行依赖管理, Apache HttpClient进行数据爬取,Jsoup对爬取到的数据解析处理.前端采用 Thymeleaf进行页面解析, BootStrap进行页面渲染. 建议使用IDEA导入项目
